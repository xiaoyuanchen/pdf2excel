# pdf2excel
**功能**：实现pdf文件内表格转化成excel文件

**源码文件**：`pdf2excel.py` 


**win系统exe程序**：`pdf2excel.exe`  
**使用方式**：本地新建一个文件夹，将`中华人民共和国海关进口货物报关单-模板.pdf`和`pdf2excel.exe`放进去，双击运行`pdf2excel.exe`程序，片刻后便得到结果文件`result.xlsx`
